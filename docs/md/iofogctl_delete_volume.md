## iofogctl delete volume

Delete an Volume

### Synopsis

Delete an Volume.

The Volume will be deleted from the Agents that it is stored on.

```
iofogctl delete volume NAME [flags]
```

### Examples

```
iofogctl delete volume NAME
```
